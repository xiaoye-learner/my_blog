# MySQL数据库的使用
## 终端：
* 登录数据库：
```sh
mysql -u root -p
```

* 显示数据库列表：
```sh
show databases;
```

* 创建数据库：
```sh
create database 数据库名字;
```

* 进入/选择数据库：
```sh
use 数据库名字;
```

* 删除数据库：
```sh
drop database 数据库名字;
```

* 查询表列表:
```sh
show tables;
```

* 查询表内的所有字段：
```sh
show full columns from 表名;
```

* 退出数据库命令行：
```sh
exit;
```

* 删除表：
```sh
drop table 表名;
```
